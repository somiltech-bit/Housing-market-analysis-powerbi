# Data Dictionary

This document describes all features used in the Housing Market Analysis dataset.

| Column Name | Description |
|-------------|-------------|
| date | Date when the transaction was recorded |
| quarter | Fiscal quarter of the transaction (Q1-Q4) |
| house_id | Unique identifier for each property |
| house_type | Property type (Apartment, Villa, Farm, Townhouse, Summerhouse, etc.) |
| sales_type | Type of sale (Auction, Regular Sale, Family Sale, Other Sale) |
| year_build | Construction year of the property |
| purchase_price | Final purchase price of the house |
| %_change_between_offer_and_purchase | Percentage difference between offer price and purchase price |
| no_rooms | Number of rooms |
| sqm | Total property area in square meters |
| sqm_price | Price per square meter |
| address | Property address |
| zip_code | Postal code |
| city | City where the property is located |
| area | Specific district or neighborhood |
| region | Administrative region of Denmark |
| nom_interest_rate% | Nominal mortgage interest rate |
| dk_ann_infl_rate% | Annual inflation rate in Denmark |
| yield_on_mortgage_credit_bonds% | Mortgage credit bond yield |

## Dataset Summary

- Total Records: 100,000+
- Country: Denmark
- Domain: Real Estate / Housing Market
- Analysis Tool: Power BI
